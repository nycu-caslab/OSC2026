Welcome to Operating System Capstone, Spring 2026!#

This course aims to introduce the design and implementation of operating system kernels. You’ll learn both concept and implementation from a series of labs.

This course uses OrangePi RV2 as the hardware platform. Students can get their hands dirty on a Real Machine instead of an emulator.
